
# Estrutura de Prompt Recomendada pelo CalAI

## Estrutura

1. PAPEL: Quem o modelo é  
2. OBJETIVO: O que você quer  
3. CONTEXTO: Informações essenciais  
4. INSTRUÇÕES: Regras claras  
5. FORMATO: Como deve vir a resposta  
6. EXEMPLO (opcional): Modelo de saída  
7. AÇÃO FINAL: O que fazer agora


## Exemplo
1. Papel:
Você é um roteirista especializado em vídeos para LinkedIn.

2. Objetivo:
Criar um roteiro de 45 segundos sobre IA generativa.

3. Contexto:
O público é iniciante e quer explicações fáceis. O tom deve ser leve e motivador.

4. Instruções:
- Seja direto
- Use frases curtas
- Evite termos técnicos
- Se faltar algo, pergunte

5. Formato:
Use tópicos, com:
- Abertura
- Ponto principal
- Exemplo simples
- Fechamento

6. Exemplo:
“Abertura rápida → Explicação simples → Exemplo → Fechamento motivador”

7. Ação Final:
Agora gere o roteiro final.
